# Benefits of Object-Oriented Programming

- **Modularity**: Code is organized into classes, making large codebases easier to manage.
- **Reusability**: Inheritance lets you reuse existing code without rewriting it.
- **Scalability**: New classes and features can be added easily.
- **Maintainability**: Bugs are easier to isolate because logic is grouped into specific classes.
- **Real-world mapping**: Real entities such as `User`, `Order`, and `Product` can be modeled naturally.
- **Data hiding**: Encapsulation protects sensitive data from outside access.
- **Collaboration**: Teams can work on separate classes independently.


 Object-oriented programming is a programming paradigm that provides a means of structuring programs so that properties and behaviors are bundled into individual objects.


# What is a Class?
A class is a blueprint/template — it defines the structure and behavior, but holds no real data by itself.
